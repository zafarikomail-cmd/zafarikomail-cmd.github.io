## 🔧 Technology Decisions

### Why Hand-Written CSS & HTML?
1. **Full Control** — Every pixel is intentional
2. **Performance** — No framework bloat; critical assets load instantly
3. **Accessibility** — Semantic HTML ensures screen readers understand structure
4. **Learning** — Building from first principles reinforces web fundamentals

### Why GitHub Pages + Custom Domain?
1. **Instant Deployment** — Git push = live updates
2. **Free & Reliable** — GitHub's infrastructure
3. **Version Control** — Every portfolio update is tracked
4. **Custom Domain** — Professional branding (via Cloudflare Pages redirect)

## 🎨 Design Philosophy

### Principles
1. **Minimalist Elegance** — Remove everything unnecessary; keep what matters
2. **Semantic Web** — HTML structure reflects content meaning, not layout
3. **Accessibility by Default** — Keyboard navigation, screen reader support, color contrast
4. **Dark & Light Modes** — System preference detection; smooth transitions
5. **Performance First** — Every millisecond counts; preload critical assets

### Color Palette
- **Light Mode:** Warm, off-white background (#F7F5F0)
- **Dark Mode:** Near-black background (#0D0C0A)
- **Accent:** Contextual colors for interactive elements
- **High Contrast:** Ensures readability for all vision abilities

### Typography
- **Headings:** Fraunces (serif, elegant)
- **Body Text:** Inter (modern, legible)
- **Code/Tech:** JetBrains Mono (monospace precision)
